
# Exam Roll No. Slip Maker

We can create bulk Exam Roll No. Slip 
Data Taken from Excel Sheet


## API Reference

#### Get all Exam Slip

```http
  GET http://localhost:3000// 
```


#### Get Single Slip Using Roll no

```http
  GET http://localhost:3000//id
```
